# IGM Theme System - Guide

## ✨ نظام الثيم الجديد - دليل الاستخدام

### المميزات الجديدة:
- 🌙 **Dark Theme** - ثيم داكن مريح للعين
- ☀️ **Light Theme** - ثيم أبيض فاتح أصلي
- 🔄 **Theme Toggle** - تبديل سريع بين الثيمات
- 💾 **Auto-Save** - الاختيار يُحفظ تلقائياً في المتصفح

---

## استخدام الميزة:

### في الصفحات الرئيسية:
انقر على زر **"Dark/Light"** في الـ Navigation Bar (الشريط العلوي)

### في صفحات الإدارة:
انقر على نفس الزر في قسم الـ User Section

---

## التفاصيل التقنية:

### الملفات المعدّلة:
- ✅ `style.css` - إضافة CSS variables للـ dark theme
- ✅ `header.php` - إضافة زر toggle
- ✅ `js/theme-toggle.js` - JavaScript للتحكم بالـ theme
- ✅ جميع ملفات الـ admin pages - إضافة الـ script والـ button

### كيفية العمل:
1. الـ button يقرأ القيمة الحالية من `localStorage`
2. عند الضغط، يتم تبديل الـ class على `<body>`
3. الـ CSS variables تتغير تلقائياً
4. الاختيار يُحفظ في المتصفح

### متغيرات CSS المستخدمة في Dark Theme:
```css
--primary: #0f1b2e
--primary-light: #1a3a5c
--text-primary: #e8eef7
--text-secondary: #b8c5d6
--bg-body: #0d1620
--bg-card: #141d2e
--border-color: #2a3f52
```

---

## الألوان:

### Light Theme:
- الخلفية: أبيض فاتح بتدرج أزرق خفيف
- النصوص: أزرق داكن
- البطاقات: أبيض نقي

### Dark Theme:
- الخلفية: أزرق داكن جداً (حوالي #0d1620)
- النصوص: أبيض فاتح (حوالي #e8eef7)
- البطاقات: أزرق داكن بدرجات مختلفة
- Hover effects محسّنة للعتمة

## استكشاف الأخطاء:

إذا لم يعمل الـ toggle:
1. تأكد من تحميل ملف `js/theme-toggle.js`
2. افتح Browser Console (F12) وتحقق من الأخطاء
3. امسح `localStorage` وحاول مرة أخرى

```javascript
// لمسح الـ localStorage:
localStorage.clear();
location.reload();
```
